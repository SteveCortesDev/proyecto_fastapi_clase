## 📖 Descripción

**FastAPI - Sistema de Gestión** es una API RESTful desarrollada con **FastAPI** que permite gestionar de manera eficiente:

- 🧑‍💼 **Clientes**: Crear, leer, actualizar y eliminar clientes
- 📄 **Facturas**: Gestionar facturas asociadas a clientes
- 💰 **Transacciones**: Administrar transacciones de cada factura

Esta API fue diseñada siguiendo los principios de **Clean Code** y **RESTful API**, proporcionando una documentación interactiva automática con **Swagger UI**.

---

## ✨ Características

- ✅ **CRUD Completo** para Clientes, Facturas y Transacciones
- ✅ **Validación de datos** con Pydantic
- ✅ **Documentación automática** con Swagger UI y ReDoc
- ✅ **Manejo de errores** con HTTPException
- ✅ **Relaciones entre entidades** (Cliente → Factura → Transacciones)
- ✅ **Cálculo automático** del valor total de facturas
- ✅ **Estructura modular** y escalable
- ✅ **Fácil de extender** y mantener
